# Urban Noise Pollution Analytics 🏙️🔊

An interactive Power BI dashboard analyzing urban noise pollution trends, temporal variations, and regulatory compliance across major cities in India, powered by an automated R data pipeline.

### Tech Stack
* **Data Processing & Automation:** R, `dplyr`, `lubridate`
* **Business Intelligence & Visualization:** Power BI, DAX
* **Data Source:** Central Pollution Control Board (CPCB) Noise Monitoring Data

### Key Features
* **Automated Data Pipeline:** Engineered an R script to merge station metadata with monthly recordings, handle missing values, and programmatically calculate daytime/nighttime exceedance metrics based on CPCB standards.
* **Interactive Dashboarding:** Built a comprehensive Power BI report utilizing DAX measures to track average noise levels and maximum noise events.
* **Compliance Tracking:** Visualized the proportion of compliant vs. non-compliant monitoring stations across Residential, Commercial, Industrial, and Silence zones.
* **Temporal Analysis:** Mapped historical noise level trends to identify high-risk urban zones and assess the effectiveness of noise mitigation over time.
